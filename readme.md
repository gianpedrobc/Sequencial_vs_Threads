# Threads em Python

Este projeto demonstra o funcionamento de **Threads** em Python utilizando o módulo `threading`.

## Objetivos

- Compreender o conceito de Threads.
- Comparar execução sequencial e concorrente.
- Demonstrar a criação de múltiplas Threads.
- Utilizar `start()` e `join()`.
- Medir o tempo de execução em cada abordagem.

## Tecnologias

- Python 3
- threading
- urllib.request
- time
- datetime

## Como executar

```bash
python main.py
